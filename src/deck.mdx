## [chrilves/elm-io](https://github.com/chrilves/elm-io)

"Elm Architecture without `update`"

</Box>

<Appear>

- update `model` in `view`:

<Box>

```
view : Model -> Html (IO Model Msg)
view model =
  div [] [
    div [] [text (String.fromInt model)],
    button [onClick increment] [text "+1"],
    button [onClick reset] [text "reset"]
  ]
```

</Box>

<Box>

```elm
increment : IO Model Msg
increment = IO.modify ((+) 1)

reset : IO Model Msg
reset = IO.set 0
```

</Box>
